1. Kündigung des PointerScore-Abonnements

Hier können Sie Ihr PointerScore-Abonnement kündigen. Das PointerScore-Abonnement ist monatlich kündbar. Die Kündigung wird zum Ende des bereits bezahlten Abrechnungszeitraums wirksam. Nach Ablauf des bezahlten Abrechnungszeitraums endet der Zugriff auf die kostenpflichtigen Funktionen von PointerScore. Bereits gezahlte Entgelte werden grundsätzlich nicht anteilig erstattet, soweit keine gesetzlichen Ansprüche bestehen.

2. Kündigung über das Benutzerkonto

Sofern diese Funktion verfügbar ist, können Sie Ihr Abonnement direkt im Benutzerkonto kündigen. Beispiel: Benutzerkonto → Abonnement → Abo kündigen Nach erfolgreicher Kündigung erhalten Sie eine Bestätigung per E-Mail.

3. Kündigung über diese Seite

Falls die Kündigung im Benutzerkonto nicht möglich ist oder Sie diese Seite verwenden möchten, können Sie Ihr Abonnement über das folgende Kündigungsformular kündigen. Bitte geben Sie die Daten so an, dass wir Ihr Abonnement eindeutig zuordnen können.
